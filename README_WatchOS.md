# ⌚️ Counting App for Apple Watch

**Counting for Watch**는 아이폰을 꺼내지 않고도 손목 위에서 즉시 숫자를 카운팅할 수 있는 컴패니언 앱입니다. 독립적인 사용이 가능할 정도로 강력하며, 아이폰과 끊김 없는 연결 경험을 제공합니다.

## ✨ 주요 특징 (Key Features)

### 1. ⚡️ 빠른 접근 및 카운팅 (Quick Counter)
- **즉각적인 실행**: 앱 실행과 동시에 아이폰에서 설정한 카테고리 목록을 확인할 수 있습니다.
- **간편한 입력**:
    - **FullScreen Tap**: 화면 어디를 터치해도 카운트가 올라가는 직관적인 인터페이스를 제공합니다 (카운터 상세 화면).
    - **Haptic Feedback**: 숫자가 올라갈 때마다 손목에 기분 좋은 햅틱 피드백을 전달하여 보지 않고도 입력 확인이 가능합니다.
    - **Animation**: 숫자가 커졌다 작아지는 쫀득한 애니메이션 효과를 적용했습니다.

### 2. 📡 실시간 양방향 동기화 (Bi-directional Sync)
- **스마트 병합 (Smart Merge)**: 아이폰과 워치 양쪽에서 동시에 수정하더라도 타임스탬프 기반으로 최신 데이터를 안전하게 합칩니다.
- **백그라운드 전송**: 앱을 켜두지 않아도 데이터가 백그라운드에서 동기화됩니다 (WCSession 활용).
- **상태 인디케이터**: 상단 헤더에 동기화 중임을 알리는 애니메이션 아이콘이 표시됩니다.

### 3. 💾 오프라인 모드 & 영구 저장 (Persistence)
- **독립 작동**: 아이폰이 근처에 없거나 인터넷이 연결되지 않아도 문제없습니다.
- **로컬 데이터 저장**: 카운팅한 데이터는 워치 내부에 안전하게 저장되며, 연결이 복구되는 즉시 아이폰으로 전송됩니다.
- **Empty State 대응**: 데이터가 없을 경우 친절한 안내 문구를 표시하여 사용자를 아이폰 설정으로 유도합니다.

### 4. 🎨 일관된 디자인 경험
- **Dark Mode Optimized**: 워치의 OLED 화면에 최적화된 트루 블랙(True Black) 배경과 선명한 포인트 컬러를 사용했습니다.
- **Digital Crown**: 디지털 크라운을 돌려 정밀하게 스크롤하거나 값을 조절할 수 있습니다 (지원 예정).

## 🛠 기술 스택 (Tech Stack)
- **Platform**: watchOS 10+
- **Framework**: SwiftUI
- **Connectivity**: WatchConnectivity (UserInfo & Message Transfer)
- **State Management**: Swift Observation (@Observable)

## 🚀 사용 가이드 (User Guide)

1. **설치**: 아이폰에 Counting 앱을 설치하면 워치에도 자동으로 설치됩니다 (설정에 따라 다름).
2. **카테고리 동기화**:
   - 워치 앱에서는 새로운 카테고리를 만들 수 없습니다.
   - 아이폰 앱에서 먼저 카테고리와 카운터를 생성해주세요.
   - 잠시 후 워치 앱을 실행하면 목록이 자동으로 나타납니다.
3. **카운팅**:
   - 원하는 카운터를 터치하여 상세 화면으로 진입합니다.
   - **화면을 탭(Tap)**하면 숫자가 1씩 증가합니다.
   - **하단의 '-' 버튼**을 누르면 숫자가 감소하고, **'초기화' 버튼**으로 0으로 되돌릴 수 있습니다.
4. **동기화 확인**: 워치에서 숫자를 올린 후 아이폰 앱을 켜보세요. 숫자가 마법처럼 바뀌어 있을 겁니다!
